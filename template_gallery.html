安装指引

前提条件

确保已安装 Obsidian 及 Templater 插件。

步骤一：复制模板文件

# 将 templates/ 目录复制到 Obsidian 库的模板文件夹

cp -r templates/* /path/to/your/vault/_templates/

步骤二：配置 Templater

在 Obsidian 设置 → 社区插件 → Templater 中配置：

  • 设置 Template folder location 为 _templates
  • 设置 Script files folder location 为 _scripts（如需要脚本）
  • 建议绑定快捷键：设置 → 快捷键 → 搜索 Templater: Insert Template

步骤三：使用模板

按快捷键 Alt+N（或你绑定的快捷键）→ 选择模板 → 自动生成。

可选：CSS 美化

将 css/ 目录下的文件复制到 .obsidian/snippets/，然后在设置 → 外观 → CSS 片段中启用。
