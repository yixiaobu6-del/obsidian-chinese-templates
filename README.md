## 笔记不再"从空白开始"

- **100个模板**：日记/笔记/项目/复盘等场景
- **评分系统**：1-5星评分，localStorage持久化
- **模板组合器**：多选模板→合并预览→一键复制
- **搜索筛选**：按分类/关键词/标签搜索

## 怎么用

打开 `template_gallery.html` → 浏览/搜索模板 → 复制使用
