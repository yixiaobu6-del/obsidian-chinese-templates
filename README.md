# Obsidian中文模板库

Obsidian笔记不知道怎么写？
**100个模板帮你快速上手。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()
[![Obsidian](https://img.shields.io/badge/Obsidian-兼容-purple)]()

## 笔记不再"从空白开始"

- **100个模板**：日记/笔记/项目/复盘等场景
- **评分系统**：1-5星评分，localStorage持久化
- **模板组合器**：多选模板→合并预览→一键复制
- **搜索筛选**：按分类/关键词/标签搜索

## 怎么用

打开 `template_gallery.html` → 浏览/搜索模板 → 复制使用
